# Tarea de Programación: Generador de Tarjetas Dinámicas con JS

## 🎯 Objetivo
Crear una página web que permita al usuario ingresar datos (título, descripción, color de fondo) y generar una tarjeta dinámica en tiempo real usando solo HTML, CSS y JavaScript.

---

## ✅ Requisitos

### 1. **HTML**
- Un formulario con:
  - Input para el **título**.
  - Input para la **descripción**.
  - Input de tipo `color` para elegir el color de fondo.
  - Botón "Crear Tarjeta".
- Un contenedor (`<div id="cards">`) donde se mostrarán las tarjetas generadas.

### 2. **CSS**
- Estiliza las tarjetas con:
  - Borde redondeado.
  - Espaciado interno.
  - Transición suave al cambiar el color.
  - Diseño responsive (que se vea bien en móviles y escritorio).

### 3. **JavaScript**
- Escucha el evento `submit` del formulario.
- Crea una nueva tarjeta con los datos ingresados.
- Añade la tarjeta al contenedor `#cards`.
- Permite eliminar una tarjeta con un botón "Eliminar".

---

## 📌 Bonus (nivel medio avanzado)
- Añadir un botón "Limpiar todo" que elimine todas las tarjetas.
- Guardar las tarjetas en `localStorage` para que persistan al recargar la página.

---
