Descarga automáticamente los 3 adjuntos (factura, comprobante, acuse_electronico) de los correos de Starlink desde avdmail@avdinternacional.com y los guarda en la carpeta de descargas del usuario.

Uso:
    python run.py descargar --recientes
    
También se puede usar para descargar todos los adjuntos:
    python run.py todos
    
Y limpiar registros procesados:
    python run.py limpiar
