# DESIGN.md

## Sistema de Colores
- **Primario**: `#1a73e8` (Azul profesional)
- **Secundario**: `#34a853` (Verde confiable)
- **Neutro Oscuro**: `#202124`
- **Neutro Claro**: `#f8f9fa`
- **Error**: `#ea4335`
- **Éxito**: `#1aa368`

## Tipografía
- **Principal**: `Inter, sans-serif`
- **Títulos**: `700` (Negrita), `1.2em` de altura de línea
- **Cuerpo**: `400` (Regular), `1.5em` de altura de línea
- **Tamaño base**: `16px`

## Espaciado
- **Margen interno**: `24px`
- **Espacio entre elementos**: `16px`
- **Padding de botones**: `12px 24px`
- **Rounded corners**: `8px`

## Componentes UI
- **Botón principal**: fondo primario, texto blanco, bordes redondeados
- **Botón secundario**: borde sólido, fondo transparente, texto secundario
- **Input**: borde de `1px` sólido, fondo blanco, borde redondeado `6px`
- **Card**: fondo blanco, sombra ligera, borde redondeado `12px`, padding `20px`

## Reglas de Accesibilidad (WCAG)
- Contraste mínimo: 4.5:1 para texto
- Tamaño de texto mínimo: `14px` para cuerpo
- Soporte para teclado (tabindex)
- Etiquetas de formulario explícitas

## Notas
- Este archivo debe actualizarse en cada cambio de diseño.
- Es el único fuente oficial de estilo para agentes de IA, diseñadores y desarrolladores.
- Válido para web, móvil, y apps de escritorio.
