```

**SLIDE 2 — El concepto base**
```
Después de 1 año
de trabajo, el empleado
tiene derecho a:

14 DÍAS HÁBILES
de vacaciones pagadas.

⚠️ HÁBILES.
No calendario.
```

**SLIDE 3 — La diferencia (visual fuerte)**
```
14 DÍAS HÁBILES =
lunes a viernes
laborables consecutivos.

Sin contar:
✗ Sábados
✗ Domingos
✗ Días feriados nacionales
```

**SLIDE 4 — Ejemplo real**
```
Si las vacaciones
comienzan el lunes 5:

12, 13, 14, 15, 16,
19, 20, 21, 22, 23

PERO 18 días corridos
en el calendario.

no calendario. ⚠️)
